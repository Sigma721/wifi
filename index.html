Live Visualization

Active WiFi devices around my apartment in the last 10 minutes.

Each circle represents a unique device. The X-axis represents average signal strength and the Y-axis represents the last 10 minutes. As devices age they move towards the bottom of the screen, unless a new packet is received

Hovering over a device will show a summary of its past hour of activity at the bottom of the screen. Each circle represents data from a 5 minute interval.

Stack Overview

A Raspberry Pi in my apartment with a WiFi card running in monitor mode that can receive packets from all sources.

A Python daemon actively listens for packets stripping the headers. Every few seconds a payload is constructed from the collected headers and serialized using Protocol buffers. The payload is POSTed to a server running an API written in Go.

The server will deserialize the payload and insert the headers into a Postgres instance. Some cron jobs run in [5 min, hourly, daily, weekly] intervals to aggregate the data.
